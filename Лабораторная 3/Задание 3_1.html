Каскадные таблицы стилей

При оформлении HTML-документов приходится несколько раз прописыаать однотипные теги. Использование каскадных таблиц стилей (CSS) позволяет значительно упростить эту работу. Поддержка стандарта каскадных таблиц стилей и их программируемых элементов появилась начиная с версии 4.0 и Netscape и Internet Explorer.


Существует три основных варианта использования таблиц стилей.

Вариант первый

Описание стиля располагается непостредственно внутри тега элемента, котрый вы описываете. Это делается с помощью парментра STYLE. Этот метод нежелателен, и понятно почему: он приводит к потере одного из основных преимуществ CSS.
